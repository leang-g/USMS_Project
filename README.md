# 🎓 USMS - University Student Management System
## 📖 User Instructions & System Guide

Welcome to the **University Student Management System (USMS)**! This guide will walk you through how to run the system and use all its features step-by-step.

## 🚀 1. How to Start the System

1. Open your terminal (Command Prompt / VS Code Terminal).
2. Navigate to the project folder:
   cd USMS
3. run python main.py
4. You will see this login screen
